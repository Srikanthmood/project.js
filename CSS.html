CSS

CSS Basics:

1. Selectors: target HTML elements (.class, #id, tag)
2. Properties: define styles (color, font-size, margin)
3. Values: assign values to properties (red, 18px, 10%)
4. Rules: combine selectors, properties, and values (.header { color: blue; })

CSS Selectors:

1. Class (.header)
2. ID (#logo)
3. Tag (h1) 4. Attribute ([href])
5. Pseudo-classes (:hover, :active)
6. Pseudo-elements (::before, ::after)

CSS Properties:

1. Color and Background
2. Fonts and Text 3. Box Model (margin, padding, border)
4. Layout (display, position, float)
5. Visual Effects (opacity, transform, transition)

CSS Units:

1. px (pixels)
2. % (percentage)
3. em (relative font size)
4. rem (root element font size)
5. vw/vh (viewport width/height)
